# TaskEarn - منصة المهام والربح

منصة احترافية للربح من إنجاز المهام عبر الإنترنت.

## المميزات

1. تصميم احترافي وأنيق
2. واجهة مستخدم سهلة الاستخدام
3. نظام مصادقة آمن باستخدام Firebase
4. لوحة تحكم للمستخدمين
5. لوحة تحكم متكاملة للأدمن
6. نظام مهام كامل
7. نظام إيداع وسحب آمن
8. تصميم متجاوب يعمل على جميع الأجهزة

## التقنيات المستخدمة

- HTML5
- CSS3 (مع تصميم متجاوب)
- JavaScript (Vanilla)
- Firebase (Authentication, Firestore, Storage, Hosting)

## خطوات التنفيذ

### 1. إنشاء مشروع Firebase

1. سجل الدخول إلى [Firebase Console](https://console.firebase.google.com/)
2. أنشئ مشروع جديد
3. أضف تطبيق ويب (Web App)
4. احصل على بيانات التهيئة (firebaseConfig)

### 2. إعداد المصادقة

1. من قسم Authentication في Firebase Console
2. قم بتفعيل مصادقة البريد الإلكتروني/كلمة المرور
3. أضف المستخدم الأدمن يدوياً:
   - البريد: Hamzaaskouk95@gmail.com
   - كلمة المرور: mama dada

### 3. إعداد قاعدة البيانات

1. من قسم Firestore في Firebase Console
2. أنشئ قاعدة بيانات جديدة
3. عيّن قواعد الأمان (انظر ملف firebase-rules.md)
4. أنشئ المستندات التالية:
   - مستخدم الأدمن في مجموعة users

### 4. إعداد التخزين

1. من قسم Storage في Firebase Console
2. أنشئ دلو تخزين (bucket)
3. عيّن قواعد الأمان (انظر ملف firebase-rules.md)

### 5. إعداد الاستضافة

1. من قسم Hosting في Firebase Console
2. اتبع التعليمات لنشر الموقع

### 6. تكوين الموقع

1. انسخ ملفات المشروع إلى مجلد
2. افتح ملف `js/firebase-config.js`
3. استبدل بيانات التهيئة ببيانات مشروعك
4. افتح الموقع في المتصفح
